def collect_crg.getRunTypeAbbreviation (   pdbT,
  runT,
  scaleT

Get the abbreviation of a certain type of run.

Args: pdbT: different types due to different pdb files used, option={"crystal", "hydro"}. runT: quick or default run of MCCE, option={"quick", "def"}. scaleT: scale lj by 0.1 or don't scale lj, option={"raw", "lj01"}.

Returns: The abbreviation the run type determined by the 3 parameters above.
