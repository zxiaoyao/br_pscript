  reverse = False,
  ifile = "head3.lst",
  ofile = "head3.lst"

Change the flag of conformers in head3.lst.

According to a residue name like "ASPA0085" and the number of protons (-1, 0, 1). "fixList" is a dictionary of residue name and number of protons. For example: fixList = {"ASPA0085":0, "ASPA0212":-1}

if "reverse" is False, the conformers in the residue which have the same number of protons will be fixed. if "reverse" is True, it's opposite, only the conformers with the same number of protons are not fixed. 

Check a head3.lst line to see if it's a dummy conformer.
